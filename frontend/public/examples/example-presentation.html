Introduction

Reveal.js est une bibliothèque JavaScript pour créer des présentations HTML interactives.

  • Présentations modernes et élégantes
  • Entièrement basé sur des standards web
  • Fonctionne dans tous les navigateurs modernes
  • Complètement gratuit et open-source

Fonctionnalités

  • Navigation intuitive (clavier, souris, tactile)
  • Thèmes personnalisables
  • Transitions entre diapositives
  • Fragments pour révéler progressivement le contenu
  • Support du Markdown
  • Mode présentateur avec notes

Exemple de code

// Exemple de code JavaScript
function createPresentation(title, slides) {
  return {
    title,
    slides,
    present: function() {
      console.log(`Présentation: ${this.title}`);
      this.slides.forEach((slide, index) => {
        console.log(`Diapositive ${index + 1}: ${slide.title}`);
      });
    }
  };
}
